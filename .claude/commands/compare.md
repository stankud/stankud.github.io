Capture local screenshots of the portfolio for visual review using Playwright.

Requires a local server running on http://localhost:8000. If one isn't running, start it with `python3 -m http.server 8000` in the background first.

Run: `node scripts/compare.js`

After running, read and display all saved screenshots from `scripts/screenshots/`: local-desktop.png, local-desktop-projects.png, and local-mobile.png.
